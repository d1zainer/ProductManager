# ProductManager

Тестовое задание для IronWaterStudio — веб-приложение для управления продуктами с авторизацией и Swagger-документацией.

---

## Описание

ProductManager — это демонстрационное веб-приложение на ASP.NET Core с использованием Entity Framework Core и SQL Server.  
Приложение позволяет:

- Просматривать список продуктов.
- Добавлять новые продукты.
- Редактировать и удалять существующие продукты (только после авторизации).
- Просматривать документацию API через Swagger.
- Авторизоваться и управлять сессией через cookie.

---

## Технологии

- **Backend:** ASP.NET Core MVC, C#  
- **ORM:** Entity Framework Core
- **База данных:** SQL Server (InMemory для тестов)
- **Frontend:** Razor Pages с Bootstrap
- **Тесты:** xUnit, Fluent    
- **Документация API:** Swagger

---

## Основной функционал

1. **Авторизация**
   - Логин/Логаут для доступа к управлению продуктами.
   - Проверка ролей и прав пользователя через cookie.

2. **CRUD для продуктов**
   - Список продуктов с возможностью редактирования и удаления.
   - Добавление нового продукта через форму с валидацией.

3. **Swagger**
   - Интерактивная документация API доступна по `/swagger`.

4. **Сидирование базы**
   - При старте приложение может автоматически создать базу данных и заполнить её тестовыми данными.
